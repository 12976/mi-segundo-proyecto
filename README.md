# 🧮 Factorial GUI

Aplicación en Python con interfaz gráfica para calcular el factorial de un número.

---

## 🚀 Funcionalidad

- Introducir un número
- Calcular su factorial
- Mostrar el resultado en pantalla
- Control de errores (números negativos o no válidos)

---

## 🖥️ Tecnologías usadas

- Python
- Tkinter (interfaz gráfica)
